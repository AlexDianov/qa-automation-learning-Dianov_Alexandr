Лист задач с добавлением\удалением задач и просмотром статистики
Тесты:
2 фикстуры:
empty_todo — создает пустой список задач

todo_with_tasks — создает список с двумя готовыми задачами для тестов выполнения и удаления. Фикстуры в pytest нужны для переиспользования подготовленных объектов в нескольких тестах

позитивные тесты : корректное отображение задачи
успешное добавление задачи
выполнение задачи 
удаление задачи
негативные:
попытка добавить пустую задачу
завершить задачу по неверному индексу
удалить задачу по неверному индексу
